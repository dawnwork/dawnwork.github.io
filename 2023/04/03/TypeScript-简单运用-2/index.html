类型推论与类型断言

在 TypeScript 中，类型推论和类型断言是两个重要的概念。类型推论可以帮助开发人员在一定程度上省略类型注解，提升开发效率。而类型断言则可以帮助开发人员更明确地指定一个值的类型。

枚举

枚举是 TypeScript 提供的一种特殊数据类型，它表示一组明确的可选值，定义一组命名常量。

功能类似于字面量 + 联合类型的组合功能，表示一组明确的可选值，定义一组命名常量 使用 enum 关键字定义枚举，枚举名称约定以大写字母开头，多个值使用 , 分隔

枚举默认是从0开始自增的数，这种被称为数字枚举。
当然也可以给枚举初始化值使用 =, 字符串是没有自增长行为的，所以如果是字符串则每个都需要初始化值

any 类型 与 typeof 操作符

any 类型

任何类型都可以被赋值给 any 类型，也可以使用 any 类型来声明变量或参数，any 类型可以包含任何值，可以通过编译检查。

但是，过多使用 any 会失去TS类型保护的优势，类型检查器会认为这些变量或函数参数可以包含任何值，不能提供类型保护。

原则：不推荐使用 any，这会失去TS类型保护优势，尽可能的避免使用 any 类型和隐式any，这两种情况都应该手动提供类型

  1. 声明变量不提供默认值并且不指定类型
  2. 函数形参不指定类型
